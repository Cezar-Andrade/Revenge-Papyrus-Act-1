# Revenge Papyrus Act 1 Concept Fangame

This is a game made in 4 days in Game Maker Studio 2, a quick simple test of concept of a fangame battle with a simple bone enemy.

# Steps to run this project:

1. Download and configure Game Maker Studio 2: https://gamemaker.io/es/download
2. Make sure you are using runtime v2024.13.1.242 if not download it from the Preferences tab (you can see that on the top right corner once Game Maker Studio 2 is running)
3. Download the project files
4. Run the .yyp file
